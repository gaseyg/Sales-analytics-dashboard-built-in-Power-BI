# Sales-analytics-dashboard-built-in-Power-BI

Интерактивный дашборд для анализа продаж розничного магазина электроники.

## Данные
- 300 транзакций за 2024 год
- 50 клиентов, 16 товаров, 4 категории
- Источник: сгенерированные тестовые данные

## Что реализовано
- Модель данных из 3 связанных таблиц (Sales, Products, Customers)
- DAX-меры: выручка, количество заказов, средний чек, доля возвратов
- Интерактивные срезы по городу и сегменту клиента
- Страница 1: общий обзор продаж
- Страница 2: детализация по товарам и городам

## Инструменты
Power BI Desktop, DAX, Power Query
